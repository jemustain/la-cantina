# La Cantina - GHC 2025 Conference Project

A Node.js landing page for the Grace Hopper Celebration (GHC) 2025 conference technology project.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and visit:
```
http://localhost:3000
```
